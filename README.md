# CyberBot — Cybersecurity Awareness Chatbot

A WPF desktop application built in C# that educates users on cybersecurity through interactive conversation, task management, a quiz game, and an activity log.

## Project Overview

CyberBot is a Cybersecurity Awareness Chatbot developed as a three-part Progressive Assessment (POE) for the IIE. It is built using **C# and WPF (Windows Presentation Foundation)** with a XAML-based graphical user interface.

The chatbot is designed to:
- Educate users on key cybersecurity topics such as phishing, passwords, malware, VPNs, and encryption
- Help users manage their personal cybersecurity tasks with reminders
- Test users' cybersecurity knowledge through an interactive quiz
- Remember user information across the conversation using a memory system
- Log all key actions taken during the session

The project was built across three parts:
- **Part 1** — Core chatbot logic: keyword recognition, sentiment detection, memory store, and audio greeting
- **Part 2** — GUI implementation using WPF/XAML with chat bubbles, a two-page layout, and dynamic responses
- **Part 3** — Advanced features: Task Assistant with JSON storage, Mini-Game Quiz, NLP Simulation, and Activity Log
- 
The chatbot answers questions on the following topics:
- Passwords
- Phishing
- Privacy
- Scams
- Malware
- Firewalls
- Encryption
- VPNs

It also provides follow-up responses, sentiment-aware replies (e.g. detecting when a user is worried or frustrated), and personalised responses using the user's name and stated interests.
